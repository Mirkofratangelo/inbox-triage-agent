# Screenshots

Drop the following screenshots into this folder. The repo's `README.md` and
`docs/setup.md` reference them by file name.

| File name             | What to capture                                                                                                                            | Used in              |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ | -------------------- |
| `n8n-canvas.png`      | The full n8n workflow canvas after import, showing all 13 nodes wired up. Zoom out so the whole graph fits.                                 | `README.md`          |
| `digest-esempio.png`  | The Google Sheet with both tabs visible — a few real rows in **Bozze da rivedere** and **Richiede te** after one run. Crop out anything personal. | `README.md`          |
| `bozza-esempio.png`   | A close-up of one row in **Bozze da rivedere**: the original message in `messaggio` and the AI-written reply in `bozza` next to it.       | `README.md`          |
| `n8n-credentials.png` | The Gmail credential settings page in n8n with Client ID / Client Secret fields filled (blur the real values).                              | `docs/setup.md`      |
| `sheet-structure.png` | The two-tab Google Sheet right after you create it, showing the header rows of both tabs but no data yet.                                   | `docs/setup.md`      |

Recommended size: 1600px wide, PNG. Use macOS `Shift+Cmd+4` or Windows
`Snipping Tool` and crop tight. If a screenshot shows real email addresses
or customer names, blur them before committing.
